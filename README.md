# Group-Regex-Waystones
## Craft Worthy

```
"d r|er of r"
```

## Group Modifyers

Rare Mob 50%+
```
"([5-9]\d|[1-9]\d{2})%.*er of r"
```

Rare Mob 30%+ and Quant 20%+
```
"([3-9]\d)%.*er of r" "([2-9]\d)%.*ant"
```

Rare Mob 30%+ and Rarity 60%+
```
"([3-9]\d)%.*er of r" "([6-9]\d|[1-9]\d{2})%.*d r"
```

Rarity 120%+
```
"(1[2-9]\d)%.*d r"
```

Quant 25%+
```
"(2[5-9])%.*quan"
```
